# CDP Next

An interactive **concept prototype** for an intergenerational knowledge‑sharing platform — built as a single, self‑contained HTML file (vanilla HTML/CSS/JS, no build step, no backend).

🔗 **Live demo:** https://vismut-hub.github.io/cdp-next/
*(goes live once GitHub Pages is enabled — Settings → Pages → deploy from `main` / root)*

---

## What it shows

CDP Next imagines how a large institution could connect graduates, HR, and senior employees around skills and mentoring. It has four linked areas:

- **Applicants** — an AI‑style quiz matches a candidate to the department where they'd fit best, then (after submitting) unlocks an **AI Senior mentor chat** with forward‑looking growth advice.
- **HR Managers** — a candidate pipeline with an honest "AI quiz answers" review, plus **The Bridge Map**: a department‑by‑department network of senior↔junior mentoring that surfaces isolated employees, turnover risk, and **stalled knowledge exchanges**.
- **Employees** — a senior‑perspective portal: mentees (buddy system), a skills exchange (teach/learn, live co‑piloting), personal interests, and anonymous monthly *pains & gains*.
- **CDP Aperitif** — a public networking‑event landing with a live countdown and registration.

> All people, candidates, and figures are **fictional sample data** for demonstration.

## Run it locally

No install, no build — just open the file:

```bash
open index.html        # macOS
# or double‑click index.html in your file manager
```

Everything (markup, styles, logic, and the SVG visualisations) lives in `index.html`.

## Tech

- Single‑file static app — plain HTML, CSS, and JavaScript
- Inline SVG for the Bridge Map network + dumbbell charts
- No dependencies, no tracking, no server
